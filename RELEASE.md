# 字段填充 v1.3.0 发布说明

## 📢 版本信息

- **版本号**: v1.3.0
- **发布日期**: 2026-03-29
- **兼容浏览器**: Chrome 90+, Edge 90+

## ✨ 主要特性

### 核心功能

- **多 Profile 管理**: 支持创建多个配置文件，分别管理不同网站的字段
- **实时字段拾取**: 鼠标悬停自动识别表单字段，无需手动编写选择器
- **智能解析**: 自动识别文本中的字段值，支持别名匹配
- **导入导出**: 全量备份/恢复配置，单个 Profile 导入导出
- **暗黑模式**: 支持跟随系统或手动切换主题
- **右键菜单控制**: 可开启/关闭右键菜单中的字段填充功能
